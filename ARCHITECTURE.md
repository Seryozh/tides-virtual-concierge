## API Endpoints

### 1. `/api/transcribe`
**Purpose:** Convert audio to text using Whisper

**Request:**
```http
POST /api/transcribe
Content-Type: multipart/form-data

audio: File (WebM audio blob)
```

**Response:**
```json
{
  "text": "Do I have any packages?"
}
```

**Latency:** ~1-2 seconds

---

### 2. `/api/chat`
**Purpose:** Process user input with AI and tools

**Request:**
```http
POST /api/chat
Content-Type: application/json

{
  "messages": [
    {
      "role": "user",
      "parts": [{ "type": "text", "text": "Do I have any packages?" }]
    }
  ],
  "sessionId": "uuid-here",
  "unitNumber": "101"
}
```

**Response:**
```
Stream of text chunks (TextStreamResponse)
```

**Features:**
- Loads conversation history (last 10 exchanges)
- Executes tools (checkPackages, logPickup, bookAmenity)
- Saves conversation after streaming completes
- Edge runtime for <200ms first-byte

**Latency:** ~500ms

---

### 3. `/api/synthesize`
**Purpose:** Convert text to speech using ElevenLabs

**Request:**
```http
POST /api/synthesize
Content-Type: application/json

{
  "text": "Yes, you have 2 packages from Amazon and FedEx."
}
```

**Response:**
```
audio/mpeg (MP3 file)
```

**Settings:**
- Voice ID: `yoEMsY6awhlIEknBgmfX` (your custom voice)
- Model: `eleven_turbo_v2_5` (optimized for speed)
- Stability: 0.5
- Similarity Boost: 0.75

**Latency:** ~198ms (tested!)
